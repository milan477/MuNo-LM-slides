Motivation
The Problem

Music understanding is hard for ALMs

What is played

Score: melody, harmony, phrasing, dynamics, form, voicing, articulation — the composer's intent.

How it is played

Performance: precise onset/offset times, velocities, rubato, expressive shaping — the performer's interpretation.

Existing ALM training pipelines rely on LLM-synthesized captions and metadata that are coarse and weakly time-grounded — limiting fine-grained musical reasoning.

→

No existing representation unifies both score semantics and performance timing in an LLM-readable format.

Background
Existing Symbolic Formats

Each format has a critical gap

ABC Notation

Compact, LLM-friendly. But no performance detail — no dynamics, no timing, no rubato.

compact no performance

MusicXML

Rich score semantics, but bloated with layout markup. Exceeds LLM context windows for full works.

expressive too verbose

MIDI

Captures timing and velocity. But no score structure — voicing, phrases, and harmony are all implicit.

timing no structure

The gap: compact + semantically expressive + time-aligned. None of the three achieves all three.

Pipeline
Caption Generation

From score + audio to time-grounded QA

①

Data Alignment

(n)ASAP: 1,067 recordings, 236 scores. Note-level alignment via DTW.

②

MuNo-LM Translation

MIDI timing + velocity merged into MusicXML → MuNo-LM.

③

Automated Captioning

Stage 1: GPT-5.5 / Gemini generate time-grounded auditory analyses.
Stage 2: Claude Sonnet converts to verified QA pairs.

→

Output: short-form, timestamp-anchored QA pairs — verifiable by exact match.

Rachmaninoff, Prelude Op.23 No.4

00:00.00

The piece opens in near-silence at 00:00.00 with a low D2 beginning a long, blurred arpeggiated resonance. The first sonority is not attacked as a block: D2, A3, D4, A4, D4, A3, F♯3, A2, and D4 are released into one sustained pedal-like wash through 00:04.22. The touch is very soft, but the second upper D4 around 00:02.30 is slightly more present, so the arpeggio swells gently from inside rather than remaining flat. At 00:04.18 the same D-major field returns — the inner notes more withdrawn — thinning toward 00:07.35, preparing the first melodic entrance.

At 00:08.18 the melody appears clearly on F♯4, held over a quiet continuation of the D-based arpeggiation.

Example generated question

"A clear melodic statement projects above a still-soft arpeggiated D harmony for the first time. At what timestamp does this melodic note first appear?"

Answer

00:08.18
